## Features

### Core Functionality
- **Lead Management** - Track all seller prospects with status updates
- **Buyer Database** - Manage your cash buyer list with preferences
- **Deal Pipeline** - Track contracts through closing stages
- **Analytics Dashboard** - Real-time KPIs and conversion metrics
- **Automatic Saving** - All data persists across sessions
- **CSV Export** - Download your data anytime
- **Search & Filter** - Find leads quickly

### Business Intelligence
- Path to 6-figures tracking ($100K annual revenue goal)
- Deal velocity metrics
- Lead-to-close conversion rates
- Revenue forecasting
- Pipeline stage analytics

## Quick Start

### Option 1: Use with React (Recommended)
```bash
# Clone the repository
git clone https://github.com/demolized/wholesaling-command-center.git
cd wholesaling-command-center

# Install dependencies
npm install react react-dom lucide-react

# Run with your React setup
npm start
```

## Usage Guide

### Adding Leads
1. Click the "Leads" tab
2. Click "Add Lead" button
3. Fill in property and seller details
4. Set lead status and save

### Managing Buyers
1. Navigate to "Buyers" tab
2. Add cash buyers with budget and criteria
3. Track last contact dates
4. Match properties to buyer preferences

### Tracking Deals
1. Go to "Deals" tab
2. Move leads through pipeline stages:
   - New Lead -> Contacted -> Qualified -> Under Contract -> Closed
3. Track assignment fees and closing dates

### Analytics
- View total leads, qualified prospects, and closed deals
- Monitor revenue progress toward 6-figure goal
- Analyze conversion rates at each stage
